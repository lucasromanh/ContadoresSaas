# ContadorPro (ContadoresSaas)

Proyecto frontend base generado para ContadorPro — sistema profesional para estudios contables.

Stack principal:

- Vite + React + TypeScript
- TailwindCSS
- TanStack Query
- Zustand
- Axios
- Recharts + Tremor
- lucide-react (íconos)
- react-hot-toast (toaster)

Instalación (Windows PowerShell):

```powershell
npm install
npm run dev
```

Notas:

- La variable de entorno para el backend se define en `.env` como `VITE_API_BASE_URL`.
- Este repositorio incluye la estructura solicitada bajo `src/` con páginas y componentes base (placeholders) listos para implementar la lógica del negocio.
